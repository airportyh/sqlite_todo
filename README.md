# Simple SQLite TODO App

This is a very simple command-line todo application built with
Python and SQlite.

## Example Usage

```
$ ./todo.py add 'Cook dinner'
$ ./todo.py add 'math homework'
$ ./todo.py list
1. Cook dinner ▢
2. math homework ▢
$ ./todo.py done 2
$ ./todo.py list
1. Cook dinner ▢
2. math homework ✓
$ ./todo.py remove 2
$ ./todo.py list
1. Cook dinner ▢
```
